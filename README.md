# ModuleBan

Landing page estática e mockup para o plugin ModuleBan — Kanban em pop-up.

## Arquivos principais

- public/index.html — página principal
- public/css/styles.css — estilos
- public/js/main.js — scripts
- .env — exemplo de variáveis de ambiente

## Visualizar localmente

1) Servidor estático com Python:

```bash
cd ModuleBan/public
python -m http.server 8000
# abrir http://localhost:8000
```

2) Usando npx serve:

```bash
cd ModuleBan
npx serve public
```

3) Abrir o arquivo diretamente no navegador:

```bash
# abrir public/index.html no navegador
```

## Variáveis de ambiente

Exemplo de `.env` (arquivo na raiz do repositório):

```env
REACT_APP_KEY=COLE_O_TOKEN_AQUI
```
